# Student Record Management System (Console-Based)

This is a console-based Python project for managing student records.
The project is created for learning Python basics.

## Features
- Add student records
- View all students records
- Search student records through student ID
- Update Any student detail through student ID
- Delete student records through student ID
- save student data
- Menu-driven console application

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Run the file:
   python main.py
